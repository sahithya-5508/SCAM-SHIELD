# 🏗️ System Architecture

---

The Scam Shield system follows a streamlined workflow to analyze suspicious messages and provide real-time security insights. Users enter or paste a message into the application, where it undergoes intelligent text analysis to identify scam-related keywords, phishing patterns, and emotional manipulation techniques. Based on the analysis, the system calculates a scam probability score and evaluates the sender's trust level. If a high-risk threat is detected, an emergency guardian alert can be triggered, while personalized safety recommendations are presented to help users make informed decisions. All processing is performed locally, ensuring fast, private, and offline scam detection.
