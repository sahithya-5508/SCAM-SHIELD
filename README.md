# 📖 About

Scam Shield is a desktop application built with **Python** and **Tkinter** that helps users identify phishing attempts, fraudulent messages, fake banking notifications, OTP scams, and other online threats before they become victims.

Unlike traditional spam filters, Scam Shield analyzes the **content and psychological intent** of suspicious messages while keeping user data completely private through offline processing.

---

# 🚀 Features

### 🔍 Smart Scam Detection
- Detects phishing and scam messages instantly
- Identifies suspicious keywords and scam patterns
- Recognizes fake banking and payment requests

### 📊 Scam Probability Meter
- Calculates a scam risk score from **0–100%**
- Categorizes messages as:
  - 🟢 Safe
  - 🟡 Suspicious
  - 🔴 Dangerous

### 🧠 Emotional Pressure Detection
Detects psychological manipulation techniques such as:

- Urgency
- Fear
- Fake authority
- Rewards
- Financial pressure

---

### 👤 Trusted Sender Analysis

Evaluates sender credibility and assigns a trust level:

- ✅ Trusted
- ⚠ Unknown
- ❌ Risky

---

### 🚨 Emergency Guardian Alerts

For highly dangerous messages, Scam Shield can:

- Warn the user immediately
- Notify a trusted guardian
- Reduce the risk of financial fraud

---

### 🔒 Offline Protection

- No internet required
- Privacy-first detection
- No cloud processing
- Faster response time

---

### 💻 Simple Desktop Interface

Built using Tkinter with an intuitive interface for users of all experience levels.

# 💡 Usage

1. Launch the application
2. Paste the suspicious message
3. Click **Analyze**
4. View:
   - Scam Probability
   - Risk Level
   - Trusted Sender Score
   - Emotional Analysis
5. Follow the recommended safety actions
